Aide administrateur en ligne

En tant qu'administrateur, vous trouverez ici des informations utiles à la bonne utilisation de cette application.

Comment utiliser l'outil administrateur ?

La page d'accueil de l'outil administrateur montre les activités déja créees par les administrateurs et leurs champs/propriétés importants.

Il est possible de modifier une activité pour mettre à jour les données ou même la supprimer. Attention : la suppression d'une activité supprime l'avancement de tous les utilisateurs ayant commencé l'activité

Pour la recherche d'une activité utiliser la combinaison "Ctrl+F" qui fait apparaitre un champ de recherche.

Le bouton créer une activité permet d'utiliser un formulaire de saisie afin d'ajouter du contenu au site en remplissant un formulaire comprenant comme contraintes :

  • Les champs avec une étoile au nom doivent être obligatoirement remplis tandis que les autres sont optionnels.
  • Le domaine est choisi parmi une liste prédéfinie.
  • L'activité représente une action à effectuer (Exemple : faire des papiers d'identité).
  • Le lieu (Ex : Hotel de Ville de Lyon).
  • Le lien vers la photo propose d'inclure une image pour l'activité
  • L'importance est un nombre entre 1 et 5. 1 est une activité obligatoire, devant être effectuée par default (Ex : trouver un logement). De 2 (activités vitales) à 5 (activités peu importantes), on trouve toutes les autres activités (Ex : 2 pour trouver un travail, 3 sortir, 5 pour apprendre à cuisiner).

Il est important de choisir un nom d'activité explicite car la recherche d'activité se base sur les noms d'activités

Le bouton annuler permet de revenir à la page d'affichage des activités. Toutes les données rentrées seront alors perdues.

Comment gérer mon compte administrateur ?

La page d'accueil de l'outil administrateur montre les activités déja créees par les administrateurs et leurs champs importants. Il est possible de modifier une activité pour mettre à jour les données ou même la supprimer en cas d'erreur

Le bouton créer une activité permet d'ajouter du contenu au site en remplissant un formulaire comprenant entre autres :

  • Le domaine, choisi dans une liste prédéfinie
  • L'activité, représentant une action à effectuer (Exemple : faire des papiers d'identité)
  • Le lieu, soit physique (Ex : Mairie du 7ème Arrondissement), soit le site internet (Ex : http://www.adsea69.fr)
  • Le lien vers la photo propose d'inclure une image pour l'activité
  • L'importance est un nombre entre 1 (activités vitales) et 5 (activités peu importantes)

Les champs avec une étoile au nom doivent être remplis tandis que les autres sont optionnels

Mettre -- dans les champs requis lorsque l'information est inconnue ou indisponible

Le bouton annuler permet de revenir à la page d'affichage des activités
